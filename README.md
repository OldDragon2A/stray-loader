# stray-loader
A loading bar for StrayMaverick.

## Keyboard Shortcuts

| Key | Description |
| --- | ----------- |
| `O` | Open the options. |
| `N` | Move to the next action. |
| `U` | Update the actions using the textarea in the options. |
